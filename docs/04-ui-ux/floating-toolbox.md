# 浮動工具箱 (HUD)

浮動工具箱是展示過程中的主要互動介面。以緊湊的 HUD 面板形式出現在游標附近。

## 啟用方式：按住操作模型

靈感來自 macOS `⌘Tab` App 切換器。

| 動作 | 輸入 | 結果 |
|------|------|------|
| 開啟工具箱 | 按住 `⌃⌥Space` | HUD 出現在游標旁，顯示 Key 清單 |
| 搜尋/過濾 | 繼續按住 + 輸入字元 | Key 清單即時過濾 |
| 快速貼上（單一結果） | 僅剩 1 個結果時放開 | Key 貼入，工具箱消失 |
| 從多個結果中選擇 | 放開 → 工具箱鎖定；用 `↑↓` + `Enter` | 按 Enter 貼入 Key，Esc 取消 |
| 依編號直接貼上 | 按 `⌃⌥[1-9]` | 立即貼入第 N 個 Key，無需工具箱 |

## 鎖定行為（方案 B）

當使用者放開按住鍵且仍有多個結果時，工具箱轉為「鎖定」狀態：

- 工具箱保持可見，**無需按住任何按鍵**
- 方向鍵導航清單
- `Enter` 確認並貼上
- `Esc` 關閉不貼上
- 直接點擊 Key 項目也可以

> 此設計防止在現場展示中意外貼上錯誤的 Key。

## HotkeyManager 按住偵測邏輯

```
keyDown → 顯示工具箱 → 監聽打字 → 轉發至搜尋欄位
keyUp → 判斷：
  - 結果 = 1 → 直接貼上
  - 結果 > 1 → 鎖定工具箱，等待使用者選擇
```
