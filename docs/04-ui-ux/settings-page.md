# 設定頁

設定頁以**獨立的 macOS 視窗**開啟（不在 Menu Bar 下拉面板中）。處理所有低頻率的配置操作。

## 設定區塊

| 區塊 | 內容 |
|------|------|
| **Key 庫** | 新增 / 編輯 / 刪除 / 分組 Key；匯入與匯出；搜尋和篩選；關聯群組管理 |
| **快捷鍵** | 自訂所有鍵盤快捷鍵；衝突偵測；每個 Key 的快捷鍵編號指定 |
| **情境模式** | 建立 / 編輯 / 刪除情境預設；每個模式的遮蔽等級和剪貼簿策略 |
| **遮蔽格式** | 依服務設定遮蔽格式；前綴長度；後綴可見性；自訂樣式 |
| **Extension 連線** | VS Code Extension 和 Chrome Extension 的連線狀態；安裝連結；同步狀態 |
| **安全性** | 剪貼簿自動清除計時器；主密碼 / 生物辨識鎖定；Accessibility 權限狀態 |

## 智慧 Key 擷取

### 擷取流程

1. **掃描**：掃描頁面內容 / 剪貼簿內容中的已知 Key 樣式
2. **識別**：根據 Key 前綴和頁面 URL 識別服務提供者
3. **確認**：顯示擷取確認對話框，附帶自動分類
4. **儲存**：使用者確認後，Key 儲存至加密本地保庫

### 確認對話框

```
偵測到 3 個 Key：
  ✓ OpenAI API Key   sk-proj-****  → 加入 OpenAI 群組？
  ✓ AWS Access Key ID AKIA****     → 加入 AWS / Production？
  ✓ AWS Secret Key    ****         ↔ 與上方建立關聯？

  [ 全部加入 ]  [ 逐一確認 ]  [ 取消 ]
```

### 關聯 Key 群組功能

| 功能 | 說明 |
|------|------|
| 順序貼上 | 一個快捷鍵依序填入兩個欄位（如 Access Key ID → Secret Key） |
| 批次匯出 | 將 Key 群組匯出為完整的 `.env` 區塊 |
| 依賴追蹤 | 當某個 Key 輪替時，提示使用者更新關聯的 Key |
