# Menu Bar App

Menu Bar App 是指揮中心。點擊圖示展開下拉面板，分為三個區域。

---

## 區域一：狀態列

顯示當前模式狀態，例如：

```
🔴 展示模式 • 錄影安全
```

附帶醒目的切換按鈕。

---

## 區域二：Key 庫（雙層階層結構）

Key 以可收合的樹狀結構組織：

### 第一層 — 服務/公司
OpenAI、AWS、Stripe、Anthropic 等

### 第二層 — 具體 Key
API Key、Secret Key、Org ID、Access Token 等

### 每個 Key 列顯示

| 元素 | 說明 |
|------|------|
| 遮蔽預覽 | 如 `sk-proj-****...8f3a` |
| 快捷鍵編號 | 如 `⌥ 1` |
| 關聯群組指示器 | 用於相關聯的 Key |

---

## 區域三：快速動作

| 動作 | 說明 |
|------|------|
| **智慧擷取當前頁面** | 觸發 Chrome Extension 掃描並匯入當前瀏覽器分頁的 Key |
| **情境模式切換器** | 下拉選單選擇當前情境預設（直播中、教學錄影、內部展示、開發中） |
| **開啟設定** | 開啟完整設定視窗 |
